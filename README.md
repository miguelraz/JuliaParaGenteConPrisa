  1. define una funcion para crear una matriz de Strang para un tamano `n`.
  ```
   2  -1  0
  -1   2 -1
   0  -1  2
   ```

   *Solucion primera*:
   ```julia-repl
        function strang(n)
           mat = [0 for i in 1:n, j in 1:n]
           for i in 1:n
               for j in 1:n
                   if i == j
                       mat[i, j] = 2
                   end
               end
           end
           for i in 1:n
               for j in 1:n
                   if abs(i - j) == 1
                       mat[i, j] = -1
                   end
               end
           end
           mat
       end
   ```
